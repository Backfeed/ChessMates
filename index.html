Quick Start

Install Meteor:

curl https://install.meteor.com | /bin/sh

Documentation is available at http://docs.meteor.com/

Clone the project:

git clone https://github.com/Backfeed/blockchess.git

cd blockchess

Go to file .meteor/packages Remove npm-container

Make initial build:

meteor

Run it

meteor
